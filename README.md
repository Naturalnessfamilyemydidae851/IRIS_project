## What is IRIS?

IRIS stands for Integrated Runtime Intelligence Service. It acts as a central hub for your artificial intelligence needs. Many people use different AI services for different tasks. IRIS connects these services into one place. You get access to many AI systems through a single application.

You can use IRIS to switch between AI providers without changing your workflow. It manages connections to popular AI platforms and custom setups. The software runs locally on your computer. It keeps your data organized and your requests efficient.

## 🛠️ Using Your AI Tools

IRIS helps you route your tasks to the right AI model. When you open the program, you will see a command line interface. This interface allows you to add or manage your AI providers.

If you have an account with a service like OpenAI, Anthropic, or Gemini, you can link it to IRIS. You can also add your own custom API addresses. Use the command `iris provider add` to include a new service.

The software sorts your requests to ensure they reach the intended target. It acts as a bridge between your computer and the powerful models on the internet.

## 🔒 Security and Privacy

IRIS cares about your data. It uses modern security standards to protect your information while it travels between your computer and the AI providers you choose.

* mTLS ensures that your connections remain private.
* Local audit logs allow you to see what IRIS does.
* No personal data leaves the machine unless you start a request.

You have full control over which providers see your information. If you stop using a provider, you can remove them from the settings of the program.

## 📈 Monitoring Performance

IRIS includes tools to help you track how your AI is performing. It watches the speed of your requests and records the status of each provider.

If a specific AI service goes offline, IRIS detects this issue. It uses a system called circuit breakers to prevent your program from crashing. This keeps your workflow smooth even when outside services struggle.

You can check your usage statistics at any time. The program displays metrics so you know exactly how much work you push through the system.

## 📋 Frequently Asked Questions

**Do I need a coding background?**
No. IRIS aims to make AI management simple. You only need to type basic commands into the provided window.

**Can I connect my own AI software?**
Yes. IRIS supports any software that follows standard AI communication rules. You can use the registry feature to store these addresses.

**How do I update the software?**
When a new version becomes available, revisit the download link. Download the new installer and run it. The setup process overwrites the old version while keeping your configuration files intact.

**What happens if a command fails?**
Check your internet connection first. If the problem continues, check your API keys for the specific provider. IRIS logs errors in the audit folder if you need to troubleshoot further.

**Is this service free to use?**
The IRIS software is free to download and use. However, some AI providers you connect to might charge money for their specific services. Check with those companies if you have questions about their pricing.

## 📖 Troubleshooting Common Issues

If the program does not open, ensure you finished the installation process. Sometimes Windows may show a security notice because the program does not have a formal publisher signature. You can click "More Info" and then "Run Anyway" if you see this message.

If you cannot connect to a specific model, verify that you entered the API key correctly. Keys are long strings of letters and numbers. Even one wrong character will prevent the connection from working.

For advanced users, you can use the `mDNS` protocol if you want to discover services on your home network automatically. This helps IRIS find local AI instances without you needing to type in the exact web address.

## 📁 Storage Locations

IRIS saves small text files that contain your settings and your history. These files usually sit in your user folder. This location ensures that your preferences save across different times you use the computer. Do not delete these files unless you wish to reset your entire configuration.
